🌱 What this is

Hatchery; Mythic Menagerie is a playable experiment.

It looks simple.

But underneath, it explores:

- movement as learning  
- systems that respond instead of instructing  
- how meaning accumulates through interaction  

🧠 Core idea

What you carry forward matters.

This is not a game about winning. It is about developing a creature.

It is a space for:

- observing  
- interacting  
- noticing what changes
  
🐉 Related System: Dragon Ring

The Mythic Menagerie is part of a larger system called Dragon Ring.

A learning environment where:

motion reveals structure

feedback replaces instruction

understanding emerges through play
